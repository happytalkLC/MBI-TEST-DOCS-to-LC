db_chat_list.json 파일 구조¶

Description¶

본 문서는 백업된 데이터에 대한 구조 설명 문서입니다.

  • 데이터 이관 파라미터는 언제든지 신규로 추가될 수 있습니다. 고객사에서는 추가 파라미터들을 감안하고 연동하시기 바랍니다.

  • Json 데이터를 Json Viewer로 사용하여 한글로 확인이 가능합니다.
